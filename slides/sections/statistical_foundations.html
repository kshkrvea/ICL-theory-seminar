When PPDs can Learn

Let \(\mathcal{S} = \{\varphi \in \Phi : \mathbb{P}_{\Phi}(\varphi) > 0\}\) be the support of the prior.
Theorem 3.1. : Under regularity conditions, there exists \(\varphi^* \in \mathcal{S}\) such that for almost every \((y, x)\)

\[ \pi(y \mid x, \mathcal{D}_n) \xrightarrow[n \to \infty]{} \varphi^*(y \mid x) \quad \text{a.s.} \] and \(\varphi^*\) is the KL-optimal approximation of \(p_0\) in \(\mathcal{S}\): \(\quad \varphi^* = \arg\min_{\varphi \in \mathcal{S}} \text{KL}(\varphi \parallel p_0)\).

-> PPD trajectory does not generally lie within the prior support S for finite sample sizes: if S is non-convex — such as a union of disjoint hypothesis classes — averaging candidate models produces a predictive function that resides outside S.
-> (A1) requires only the existence within P of a unique distribution p∗ minimizing the KLD relative to p0
-> (A2) prevents the distribution П from spreading significant probability mass across an infinite number of B_j: fundamentally distinct, statistically easily distinguishable hypotheses.

PFN Approximation of the PPD

Four factors govern how well \(q_{\widehat\theta}\) approximates the family of PPDs \(\{\pi(\cdot \mid \cdot,\, \mathcal{D}_n)\}_{n}\):

  • Model capacity \(\{q_\theta : \theta \in \Theta\}\) must be expressive enough to represent \(\pi\);
  • Hypothesis space \(\Phi\) should be complex enough;
  • Monte-Carlo size: \(\widehat\theta\) \(\rightarrow\) \(\theta^*\) at rate \(\sim m^{-1/2}\) \(\Rightarrow\) a large number of steps \(m\) is needed;
  • Prior over context sizes serves as PPD's complexity regularizer.
Mechanism (for N):
- Larger n => more versatile PPD => expect approximation to model more complex functions
- Open ?: we train q by iterating over Ns => need to see large enough Ns to have complex functions
- TabPFN was trained only on N in {1, ..., 1023} yet still generalizes well to much larger n at test time -- the next section looks at why.

Bias-Variance Decomposition of PFNs

\[\begin{aligned} q_{\theta}(y \mid x, \mathcal{D}_n) - p_0(y \mid x) \;= \end{aligned}\]
\[\begin{aligned} & \textcolor{teal}{\underbrace{\textcolor{black}{q_{\theta}(y \mid x, \mathcal{D}_n) - \mathbb{E}_{\mathcal{D}_n \sim p_0^n}\big[q_{\theta}(y \mid x, \mathcal{D}_n)\big]}}_{\textstyle\text{variance}}} &+\; \;\; \textcolor{blue}{\underbrace{\textcolor{black}{\mathbb{E}_{\mathcal{D}_n \sim p_0^n}\big[q_{\theta}(y \mid x, \mathcal{D}_n)\big] - p_0(y \mid x)}}_{\textstyle\text{bias}}} \end{aligned}\]

How does each term behave asymptotically as \(n\) grows?

-> variance term: scatter of the prediction around its average over training sets drawn from p_0.
-> bias term: offset of that average from the truth. -> empirically the error keeps decreasing with n even for sizes never seen in pre-training;
-> next three slides explain why: variance dies by architecture, bias only dies with locality.

Variance and Diminishing Sensitivity

Diminishing sensitivity: Let \(\exists \alpha > 0\), \(L < \infty\) s.t. for large \(n\) and almost all data sets \(D_n,\, D_n'\) differing in one sample, \[ \big|\, q_{\theta}(y \mid x, D_n) - q_{\theta}(y \mid x, D_n') \,\big| \;\le\; L\, n^{-\alpha}. \tag{*} \]

Theorem 5.2 + Lemma 5.3 (informal):

If (*) holds, the variance is bounded by a ratio \(\sim n^{1/2-\alpha}\).
If (*) holds with \(\alpha > 1/2\), then the variance \(\xrightarrow{n \to \infty} 0\)

-> Read (*) as a bounded-differences / stability condition: replacing one sample moves the output by at most L n^-alpha.
-> Crucially this is a property of the ARCHITECTURE \(q_\theta\) as a function of \(D_n\), valid at any \(n\) — this is the part of the error that keeps improving on context sizes the network never saw.
-> Foreshadow: transformers satisfy (*) by construction.

Bias and the Need for Locality

Theorem 5.4 (informal):
If bias \(\mathbb{E}_{\mathcal{D}_n \sim \varphi^n}\big[q_{\theta}(y \mid x, \mathcal{D}_n)\big] - \varphi(y \mid x) \xrightarrow{\;n \to \infty\;} 0\) for every truth \(\varphi \in \Phi\), predictor \(q_{\theta}\) with diminishing sensitivity \((*)\) must be local.

Definition: \(q_{\theta}\) is local if there is a sequence \({\color{#f97316}\epsilon_n} \to 0\) such that arbitrarily resampling the labels of far points \(\big(\|X^{(i)} - x\| > {\color{#f97316}\epsilon_n}\big)\) leaves the prediction unchanged in the limit: \(\;\big|\, q_{\theta}(y \mid x, \mathcal{D}_n) - q_{\theta}(y \mid x, \widetilde{\mathcal{D}}_n) \,\big| \xrightarrow{\;n \to \infty\;} 0\) a.s.

-> if the predictor is to be asymptotically unbiased for EVERY plausible truth p, it has to not rely on far-away samples
-> So asymptotically only an epsilon_n-neighbourhood of x may contribute.
-> !!! This is only a necessary condition (the constant predictor is local but biased)
-> locality means the effective sample count is o(n)

Main insights

  • With enough model capacity, a rich enough \(\Phi\) and enough Monte-Carlo steps,
    pre-training makes \(q_{\widehat\theta} \approx \pi\) for context sizes from \(\mathbb{P}_N\);
  • In practice \(\mathbb{P}_N\) is necessarily limited yet predictions keep improving on larger \(n\);
  • That happens since attention forces the variance to vanish (\(\sim n^{-1/2}\)) for any \(\widehat\theta\);
  • The bias does not vanish though; localization is a possible solution.
